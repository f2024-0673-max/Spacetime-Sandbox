# MultiverseSimiulator:

An interactive physics sandbox featuring a high-performance C++ simulation engine integrated with a dynamic, browser-based Matter.js visualizer.

## 🚀 Key Features
* **Polymorphic Celestial System:** Simulates Stars, Planets, and Black Holes using Object-Oriented modern C++ principles.
* **Custom Newtonian Gravity Engine:** Features vector mechanics adjusting gravitational pull based on mass and distance.
* **Strategy Design Pattern:** Physics engines can be swapped seamlessly dynamically at runtime.
* **Interactive Frontend:** Built with Matter.js to display orbital paths, collision rigid-bodies, and real-time canvas rendering.

## 🛠️ Tech Stack
* **Backend:** C++
* **Frontend:** HTML5, CSS3, JavaScript (Matter.js)
